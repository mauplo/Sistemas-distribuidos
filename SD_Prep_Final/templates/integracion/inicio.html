Proyecto de Integracion

Registro de ventas con una sola URL

La aplicacion recibe una venta, detecta la tienda y la envia al servicio correcto. Despues guarda el resultado en SQLite para que puedas consultar el historial y el total acumulado.

Como registrar una venta

Usa una URL con este formato:

/integracion/<tienda_id>/<monto>/recibe_ventas/

  1. Tienda 1 usa el servicio SOAP.
  2. Tienda 2 usa el servicio REST.
  3. Monto debe ser un numero decimal mayor a cero.

Total acumulado

${{ total_montos }}

Este total considera todas las ventas persistidas en la base SQLite del proyecto.
